# Poker Range Trainer

A small React + TypeScript app to help you **build and train preflop ranges** for poker study.

## Getting started

1. Install dependencies:

```bash
cd poker-range-trainer
npm install
```

2. Start the dev server:

```bash
npm run dev
```

Then open the printed `http://localhost:xxxx` URL in your browser.

## Current features

- Configure a spot by **position + action** (e.g. BTN open, SB defend vs open).
- Build a range using an **interactive hand grid**.
- Run a simple quiz: the app shows a random hand and you answer whether it belongs in your range.

This is just a starting point—you can extend it with PLO ranges, different board textures, frequencies, and more.
